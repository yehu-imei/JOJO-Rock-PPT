# JOJO & Rock PPT

荒木飞吕彦《JOJO的奇妙冒险》× 摇滚乐主题演示文稿

一个将 JOJO 角色、替身与摇滚文化结合的多媒体 PPT 项目。

## 目录结构

```
PPT/
├── JOJO&Rock.pptx           # 主演示文稿（含音频、视频嵌入）
├── illustration/            # 插画素材
│   ├── Gold Experience/     # 黄金体验（乔鲁诺·乔巴拿）
│   ├── King Crimson/        # 绯红之王（迪亚波罗）
│   ├── Killer Queen/        # 杀手皇后（吉良吉影）
│   ├── Sticky Fingers/      # 钢链手指（布鲁诺·布加拉提）
│   ├── Figures/             # 音乐人/乐队图（David Bowie, Freddie Mercury, Beatles 等）
│   ├── Album/               # 摇滚专辑封面（约 40 张）
│   └── Reference/           # 参考素材（荒木画稿、动作参考等）
├── Audio/                   # 配乐参考
│   ├── The Beatles - Hey Jude.mp3
│   ├── Killer Queen.mp3
│   ├── Gold.mp3
│   └── ...（共 5 首）
└── Video/                   # 视频素材（未上传）
```

## 内容特色

- **JOJO 替身专题**：Gold Experience、King Crimson、Killer Queen、Sticky Fingers 四大替身的视觉素材
- **摇滚文化联动**：大量摇滚乐队/音乐人专辑封面及人物素材
- **参考图集**：荒木飞吕彦原画、经典动作参考、时尚参考等

## 使用说明

1. 下载 `JOJO&Rock.pptx` 并直接用 PowerPoint / WPS 打开
2. `illustration/` 和 `Audio/` 为 PPT 引用的外部素材，如需完整体验请一并下载
